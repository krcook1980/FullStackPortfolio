Inventory Organization Utility

This inventory location and notification utility was a group project created for a local company having issues with inventory organization. Using Google Maps, with a map overlay and smtpjs.com we were able to create a simple and responsive organizational utility to help them improve productivity, efficiency, and communication. The email function uses a secure key for privacy, and the map generates a location marker based on the product information entered. This product won the class award "Best Business Solution".

Weather Dashboard

This fun weather dashboard using OpenWeatherMap.org allows you to search for a desired city in the United States to receive current weather, humidity, wind speed, and uv index info as well as the 5 day forecast. To add my flare, the page background changes based on the current weather condition listed, and I found more appealing weather icons. Previously searched cities are stored to easily revisit the information, and if you leave and return to the page the last city searched will populate.
